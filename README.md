# BoardPick-server

## 소개
보드픽은 다양한 보드게임 안에서 내가 좋아하는 보드게임을 찾고, 

관련 후기들을 한 눈에 볼 수 있는 
보드게임 정보 및 후기 제공 서비스입니다.
